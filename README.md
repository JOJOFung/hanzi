隐私政策
我们重视您和孩子的隐私。本政策用于说明本应用如何处理与练字学习相关的数据。
本地数据存储
练习本、练习记录、书写偏好等内容，默认仅保存在您的设备本地。
您无需注册账号即可使用大部分功能。
匿名使用统计
在未登录状态下，应用会向我们的服务器上传有限的匿名统计信息，包括：

• 练习本标识
• 练习时长
• 是否为订阅用户
• 订阅类型（如免费版、月订阅等）

这些信息不包含：

• 姓名
• 邮箱
• 手机号
• 设备唯一标识
• 具体练习内容
• 汉字笔迹数据

服务器仅按日期进行汇总统计，无法用于识别个人身份。

这些匿名统计仅用于：

• 改进产品体验
• 优化课程与内容
• 了解功能使用情况
• 保障应用正常运行

相关数据不会用于广告投放或个性化营销。
云端同步
当家长自愿注册并登录账号后，练习本与练习记录可同步至云端，以支持：

• 多设备同步
• 数据恢复
• 更换设备后的继续使用

账号信息（如邮箱）仅用于身份验证、登录与数据同步，我们不会出售或共享给第三方广告平台。
第三方服务
我们不接入第三方广告 SDK。
我们也不会向广告网络或数据经纪商共享孩子的笔迹、练习内容或学习记录。

如未来接入必要的基础服务（例如支付、云存储或崩溃诊断服务），我们会确保其仅在提供相关功能所需范围内处理数据。
儿童隐私
本应用面向儿童学习场景设计。
我们尽量减少数据收集，并仅收集维持核心功能运行所必需的信息。

家长可随时删除应用内数据，或停止使用云同步功能。
